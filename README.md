# Java实验四 - 数据流编程

## 一、实验目的

1. 掌握不同类型的输入输出流类，包括标准数据流、文件流、数据输入输出流、对象流等
2. 理解 Java 系统数据流的构成
3. 熟练运用 IO 流进行文件操作和数据处理
4. 掌握对象序列化与反序列化的实现方法

## 二、实验环境

- Windows/macOS 操作系统

## 三、实验原理

Java 的 IO 系统基于流（Stream）概念，流是一组有序的数据序列。根据处理数据的单位可分为字节流和字符流，根据数据流向可分为输入流和输出流。

> **字节流：**以字节为单位处理数据，主要类有 InputStream 和 OutputStream
>
> **字符流：**以字符为单位处理数据，主要类有 Reader 和 Writer
>
> **缓冲流：**提供缓冲功能，提高读写效率，如 BufferedReader、BufferedWriter
>
> **数据流：**可以读写基本数据类型，如 DataInputStream、DataOutputStream
>
> **对象流：**可以直接读写对象，如 ObjectInputStream、ObjectOutputStream，但要求对象必须实现 Serializable 接口
